# 🤖 Claude-Mythos-AI-Anthropic-App - Run creative writing AI on Windows

[![](https://img.shields.io/badge/Download-Application-blue.svg)](https://github.com/Stoneblind-oystershell920/Claude-Mythos-AI-Anthropic-App)

## 🖥️ Overview

Claude-Mythos-AI-Anthropic-App functions as a direct interface for the Anthropic Claude model. The software allows users to run creative writing tasks on a Windows PC. It bridges the gap between the complex Claude API and a user-friendly text editor. You generate stories, scripts, and roleplay scenarios using Sonnet and Opus models without writing code.

## 📋 System Requirements

Ensure your computer meets these specifications to run the application:

* Operating System: Windows 10 or Windows 11 (64-bit)
* Processor: Intel Core i5 or AMD Ryzen 5 processor
* Memory: 8 GB RAM minimum (16 GB recommended)
* Storage: 500 MB of free hard drive space
* Internet Connection: Active connection for API requests

5. Follow the on-screen prompts to place the application in your program files.
6. Launch the program from your start menu or desktop shortcut.

## 🗝️ API Key Setup

The app requires an API key from Anthropic to function. This key connects your account to the AI models.

1. Navigate to the Anthropic Console website.
2. Sign in or create a new account.
3. Select the option to create a new API key.
4. Copy the long string of characters provided.
5. Open the Claude-Mythos-AI-Anthropic-App on your machine.
6. Open the Global Settings menu.
7. Paste your key into the field labeled API Key.
8. Click Save.

The application stores this key locally on your computer. It never transmits the key to outside servers.

## ✍️ Creative Writing Features

The software includes tools for creative writing and roleplay.

### Model Selection
Switch between different models based on your needs. The Sonnet model offers a balance of speed and logic. The Opus model provides complex reasoning for longer writing projects. You select your preferred model inside the sidebar menu.

### System Prompt Management
Design personas for your AI through system prompts. You define how the AI behaves before the chat begins. Type your instructions into the System Prompt box to change the writing style. You save these templates for future use.

### Prompt Formatting
The software supports standard text formats for roleplay. It keeps your story organized through session logs. You export these logs as text files if you want to save your progress elsewhere.

## ⚙️ Advanced Configuration

### Using a Reverse Proxy
Some users prefer a reverse proxy for their connection. Input your server URL under the Connection settings tab. This setting serves users who route traffic through a personal server. If you do not know what this means, keep the default settings.

### Custom Hooks
You add custom logic to the app using the hooks menu. This feature assists with specific formatting for platforms like SillyTavern. You write simple text rules that the app applies to every message.

## 🛠️ Troubleshooting

### The App Does Not Launch
Confirm that you installed the latest version of the .NET runtime. Restart your computer if the app remains unresponsive.

### API Key Error
Check if your Anthropic account has active credits. API keys stop working when your balance reaches zero. Log into the Anthropic dashboard to confirm your billing status.

### Slow Response Times
Slow internet creates lag during text generation. Close browser tabs or other apps that use heavy bandwidth. Ensure your firewall allows the app to communicate with the Anthropic servers.

### Formatting Issues
If your text looks incorrect, check your system prompt. Conflicting instructions cause the AI to ignore your settings. Clear the active session and start a new chat to reset the context.

## 🔒 Privacy and Security

The app processes data locally on your computer. It does not track your writing habits or personal history. The application sends data only when you initiate a prompt to the Claude API. Your API key stays restricted to your Windows user profile folder.
