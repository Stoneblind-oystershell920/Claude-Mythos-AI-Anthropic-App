## Overview
**Claude Mythos APP** is a powerful, open-source desktop and mobile client for **Claude** models by Anthropic. Built with the beautiful Mythos aesthetic, it is optimized for creative writing, roleplay, deep reasoning, and long-context conversations.

> [!Important]
> This application requires an official **Anthropic API Key**. No keys are bundled — everything runs privately on your device.

## Key Features
- **Mythos Design**: Stunning dark interface inspired by the Claude Mythos aesthetic.
- **Roleplay & Creative Writing**: Excellent support for character cards, long-form stories, and immersive RP.
- **Advanced Reasoning**: Full support for Claude’s Chain-of-Thought and complex problem solving.
- **System Prompt Tools**: Easy management of custom system prompts and jailbreak styles.
- **Ultra-Fast Streaming**: Highly optimized response streaming.
- **Privacy First**: Local chat storage, encryption, and reverse proxy support.
- **Multi-Platform**: Native support for Windows, macOS, Linux, Android (APK), and iOS-ready.
- **Multi-Modal**: Image upload, document analysis, and vision capabilities.

## Tech Stack
- **Core**: Python 3.12+ / Rust (backend)
- **Frontend**: Next.js + Electron / Tauri
- **LLM Integration**: Anthropic API + LangChain

## Installation
### For Regular Users
1. Download the latest release
2. Extract and run the application
3. Enter your Anthropic API Key

### For Developers
```bash
# Clone the repository
git clone --recursive https://github.com/AbhishekK130804/Claude-Mythos-AI-Anthropic-App.git

# Install dependencies
pip install -r requirements.txt

# Run in development mode
npm run dev
# or
python main.py
```

## ⚙️ Recommended Settings
* **Temperature**: `0.7–0.85` — for creative writing & roleplay  
* **Temperature**: `0.2–0.4` — for coding and reasoning  
* **Top_P**: `0.95`  
* **Max Tokens**: `8192–200000` (depending on model)
